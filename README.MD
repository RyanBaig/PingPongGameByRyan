# Ping Pong Game
### Basically a PingPong Game
| Key     | Controls                 |
|---------|--------------------------|
| W       | Move Left Pad Upwards    |
| S       | Move Left Pad Downwards  |
| ↑ Up    | Move Right Pad Upwards   |
| ↓ Down  | Move Right Pad Downwards |

## Modules:
- [x] [Pygame](https://pygame.org/docs)
